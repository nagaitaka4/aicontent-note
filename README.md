# AIコンテンツ運用ノート

> ワクワクし続けながら、人がAIコンテンツを仕事として回していくための思考と記録

---

## このリポジトリについて

「AIコンテンツ運用ノート」は、AIを活用したコンテンツ運用・SEO・副業を検証しながら、その知見を記録・公開するメディアプロジェクトです。

コンテンツ運用代行サービスの立ち上げを軸に、以下の3つを同時に実践・発信しています。

- AIを使ったブログ記事制作の仕組みづくり
- SEOによる検索流入の獲得
- Webライター副業の月3万円検証

**最重要思想：完璧より継続。止めずに回すことを最優先とする。**

---

## ターゲット

- 個人事業主・中小企業でコンテンツ運用に課題がある人
- コンテンツを外注すべきか内製すべきか迷っている人
- AIを使ったコンテンツ運用の現実を知りたい人

## 運用ルール概要

### コンテンツ方針
- SEOを前提とした長文記事（WordPress掲載）
- 検証ベース・再現性重視・実務に落とす
- H1〜H3構造、Markdown形式で出力

### SEO方針
- ロングテールキーワードで検索意図に沿った記事設計
- 1記事1テーマ徹底
- タイトルは30〜40字、キーワードを自然に含める

### 執筆スタイル
- 現実ベース・実体験重視・誇張しない
- 具体例・数字・体験談を軸に構成
- 抽象論のみの説明は禁止

詳細は [docs/](./docs/) 以下の各ドキュメントを参照してください。

## サービスモデル

ブログ運用を通じた信頼構築から、コンテンツ運用代行の受注につなげる。

```
記事 → 信頼 → 問い合わせ → コンテンツ運用代行
```

**コンテンツ運用代行サービス概要（想定）**
- 月4〜8記事の継続納品
- 月額 80,000円〜150,000円
- 対象：個人事業主・中小企業
