# コンテンツ戦略

## コンテンツ定義

本メディアにおけるコンテンツは、主にブログ記事を指す。

・SEO前提の長文記事
・検索流入を軸とする
・資産化を目的とする

SNSや短文は補助的な役割とする。

## 記事の種類
① 副業検証シリーズ（主軸）
・AIライティング副業の実践記録
・案件応募〜作業〜結果までを記録
・収益・作業時間・難易度を数値で可視化
・成功だけでなく失敗も含める
→ 本メディアの中心コンテンツ
→ 信頼構築と集客の軸

② 検証記事
・特定テーマの検証（SEO / AI活用など）
・仮説 → 実行 → 結果の形式
・数値・変化・気づきを記録
→ 再現性のある学びを提供

③ ノウハウ記事
・検証や実体験から抽出した方法論
・テンプレ化できる内容
・初心者でも実践できる形に整理
→ 読者が「できる状態」を作る

④ 体験記事（補助）
・単発の体験・気づき
・作業ログや気づきのメモ
・シリーズ化しない内容
→ 補助的なコンテンツ

## 優先順位
1. 検証記事
2. ノウハウ記事
3. 体験記事

## 内部リンク戦略
・シリーズ記事で繋ぐ
・検証 → 結論 → CTA

## CTA方針
・自然に誘導
・押し売りしない
・「相談できる」状態を作る

## 記事分類
・検証記事（40%）
・SEO記事（40%）
・仕事記事（20%）

## 検証記事
実際に試して結果を出す

## SEO記事
検索意図を満たす

## 仕事記事
サービス導線

## 方針
・実体験ベース
・誇張しない
・再現性重視

## 副業検証シリーズ
本メディアのコンテンツとして、副業検証シリーズを展開する。

### 目的
・実体験ベースで信頼を構築する
・コンテンツ運用のリアルを伝える
・収益化プロセスを可視化する

### 内容
・AIライティング副業の検証
・クラウドワークス等での案件応募
・実際の作業時間・単価・難易度
・失敗や課題も含めて記録する

### 特徴
・リアル重視（成功だけでなく失敗も含む）
・数字・具体例を必ず入れる
・シリーズとして内部リンクで繋ぐ

### 位置づけ
・検証記事の中核コンテンツ
・集客＋信頼構築の中心

### 導線
副業検証 → ノウハウ理解 → 運用代行への関心
