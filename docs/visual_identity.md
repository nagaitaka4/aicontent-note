# ビジュアルアイデンティティ

## マスコットキャラクター

### 基本情報

| 項目 | 内容 |
|---|---|
| 名前 | AI猫編集長 |
| 役割 | このメディアのマスコット・運営者のアバター |
| 見た目 | 黒猫・サングラス・大きなペンを持っている |
| スタイル | イラスト（フラット・2D） |

### 使用方針
- 全記事のアイキャッチに入れる方向（必須ではない）
- 目立ちすぎず、画像のどこかに自然に配置する
- 押しつけがましくなく、気づいたら「いる」くらいの存在感が理想

### 配置アイデア（記事ごとに変えてよい）

| 配置パターン | 説明 |
|---|---|
| 置物・フィギュア | デスクの上に置かれた置物として登場 |
| ポスター | 部屋の壁に貼られたポスターの中に登場 |
| PC画面 | ノートPCの画面や壁紙として登場 |
| カレンダー・メモ | カレンダーやノートのロゴ・落書きとして登場 |
| ステッカー | ラップトップやノートに貼られたステッカーとして登場 |
| その他 | コーヒーカップのデザイン、本の表紙など |

## 記事種別ごとのアイキャッチ方針

### 副業検証シリーズ（side-business）
- **スタイル**：イラスト統一テンプレート（青背景）
- **運用**：タイトルテキストのみ差し替え（#1・#2・#3…と番号も更新）
- **AI猫編集長**：テンプレート内に配置済み（右側に大きく登場）
- **制作ツール**：Illustrator / Photoshop

### SEO記事・仕事記事・その他
- **スタイル**：リアル寄り写真調（Fireflyで生成）
- **AI猫編集長**：Photoshopで合成。自然な形で画像内に忍ばせる
- **制作ツール**：Firefly（背景）＋ Photoshop（猫合成・書き出し）

## ファイル命名規則

| 種別 | 命名規則 | 例 |
|---|---|---|
| 全記事共通 | `eyecatch_XXXX.png`（記事frontmatter `no` と一致） | `eyecatch_0001.png` / `eyecatch_0012.png` |

**ルール：**
- 記事のフロントマター `no` と画像番号を完全に一致させる
- シリーズ・種別にかかわらず `no` で統一管理
- 形式はすべて PNG に統一
- 番号は4桁ゼロ埋め（0001, 0002, … 0010, 0100, 1000）
