# ライティングルール

## 前提
本ルールはブログ記事（長文コンテンツ）を前提とする。

## 基本方針
・現実的であること
・誇張しない
・ワクワク感を残す
・読みやすい
・自虐を入れる
・ネガティブな表現ばかりは避ける

## NG
・煽り
・抽象論
・誇張
・再現性のない話
・AI万能論

## OK
・実体験
・具体例
・数字
・検証結果
・失敗談
・成功談
・リアルな感想
・現実的な未来予測
・実務に落とす方法

## 構成ルール
H1：タイトル（必ず記事の内容を表すもの）
H2：H1の問題提起/結論などの主要トピック
H3：具体/詳細解説（必要な場合のみ）の小見出し

## トーン
・丁寧
・フラット
・信頼感重視
・親しみやすい
・専門用語は必要に応じて説明する

## 構造
導入（見出しなし）

H2
H3（必要な場合のみ）

結論

## 文体
・シンプル
・論理的
・短文
・わかりやすく
・会話調も適度に入れる
